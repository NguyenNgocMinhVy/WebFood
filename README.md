# WebFood 

Live Demo: https://nguyenngocminhvy.github.io/WebFood/

## Project Description

WebFood is a simple food ordering website built with **HTML, CSS, and JavaScript**.
This project was created as a first-year university assignment to practice **frontend development and requirement analysis**.


## Features

* Browse food menu
* View product details
* Add items to cart
* Login page
* Contact page


## Technologies Used

* HTML
* CSS
* JavaScript
