# WebFood 

## Project Description

WebFood is a simple food ordering website built with **HTML, CSS, and JavaScript**.
This project was created as a first-year university assignment to practice **frontend development and requirement analysis**.


## Features

* Browse food menu
* View product details
* Add items to cart
* Login page
* Contact page


## Technologies Used

* HTML
* CSS
* JavaScript


## Project Structure

css/ – stylesheets
js/ – JavaScript files
picture/ – images
vd/ – videos

Main pages:

* index.html
* product.html
* cart.html
* login.html
* contact.html
